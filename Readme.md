# Jobber
AI Job Scraper &amp; Outreach Engine

## Try Demo
Request tally form link.
(Output is demo data extracted from live data. As Some api require paid API/subscriptions.)

## What it Does
- Scrapes jobs from LinkedIn, Dice, Indeed, CareerBuilder
- Finds company ( non-recruitment companies) decision-makers
- Extracts & verifies emails
- Generates personalized outreach copy
- Adds prospects to a sequencer (Instantly)
- Sends structured data to your CRM or Airtable

## How to Use
- Job Title: e.g., Sales Manager
- Country: US
- Job Portals: Select any
- Email: Any inbox to receive the output
- Remove Recruitment Firms: Yes/No
- Job Posted Within: Choose timeframe
- Instantly Campaign ID: Optional

## Tech Stack
N8N, AI Agents, OpenAI, Apollo, Airtable, MillionVerifier, No2Bounce, Apify, Instantly, Gmail, Google Sheets, Tally
